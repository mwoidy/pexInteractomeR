This repository contains R notebooks used for complementary analyses and visualizations related to the peroxisomal interactome.

All data required to generate the main figures are stored in the data/ directory. 
These data were generated using Python code available at: https://github.com/juliavcramer/PEX

Output files are written to the output/ directory. 
If you prefer a different output location, you can modify the corresponding path in the relevant section of each notebook.
